# Examples

## Show vendors list

```bash
cve-cli --all-vendors
...
```

## Show all products for vendor `apple`

```bash
cve-cli --products apple
...
```

## Show info about CVE-2010-3333

```bash
cve-cli --cve CVE-2010-3333
...
```

## Show all info related to config

```bash
cve-cli --config vendors.toml
...
```
